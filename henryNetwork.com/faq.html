    Sıkça Sorulan Sorular

    Sıkça Sorulan Soruları görüntülüyorsunuz.

    Ürünlerde bulunan lisans sistemi nedir?

    Ürünlerimizde bulunan lisans sistemi, hem sizin haklarınızı hem de bizim haklarımızı korumak amacıyla geliştirilmiştir. Satın aldığınız bir ürünü kullanabilmek için, profiliniz üzerinden ürünün çalışacağı IP adresini tanımlamanız gerekmektedir. Ürün yalnızca belirttiğiniz IP adresinde aktif olacaktır.

    Ürün ile alakalı destek alabilir miyim?

    Ürün ile alakalı her konuda destek alabilirsiniz fakat destek alanımız sadece ürün ve ürün içeriğini kapsamaktadır. Yaptığınız değişikliklerden dolayı karşılaştığınız hatalarda destek sağlayamamaktayız. Aynı şekilde, ürün içerisinde leak kullanım tespit edersek ürün ile alakalı hiçbir şekilde destek sağlamamaktayız.

    İade hakkım var mı?

    Dijital ürünler kapsamında hiçbir ürünün iade hakkı yoktur.

    cURL yazılımı ne işe yarar?

    cURL yazılımı, lisansların daha stabil çalışmasını sağlar. Makinenizi herhangi bir şekilde yormaz. Profil kısmında ücretsiz ürünlerden indirebilirsiniz.

    Paket çalışmama, içinde ki eklentilerin düzgün çalışmama sorunu neden olur?

    • Paketin açılmama veya düzgün çalışmama sorunu genellikle Java kaynaklı bir sorundur. Bu sorunu düzeltmek için paketinizin sürümüne uyumlu Java versiyonunu indirip kurmanızdır. (Oracle sitesinden x64 windows olanını indirmeniz tavsiye edilir.) Paketlerimize uyumlu Java Sürümleri: Epic SkyBlock 1.17 - Java 17 Diğer Paketler - Java 16 • Makinenizin ana dilinin İngilizce (ENG) olduğuna emin olun. • Site adresimizden Lisanslarım kısmına girip satın aldığınız ürünün lisans ip kısmına makinenizin ip adresini yazmayı unutmayın.

    Lisans nasıl aktif edilir?

    Site hesabınızın profilim kısmından Lisanslarım kısmına girip sahip olduğunuz hizmetin düzenleme menüsüne girin, girdikten sonra IP, Domain kısmına makine ip adresinizi yazıp kaydet tuşuna basın.
